Task 3 - Smooth scroll

Our navigation links are messed up! Use click events to make them scroll smoothly.

  • Attach a click event listener for each <a class="nav-link"> element inside the fixed navbar
    • Take the target to which we want to scroll to from the data-link data-attribute
    • Format this value so it can be used as a CSS selector
    • Tip: inspect the <section>-elements to figure out what kind of selector should be used
    • Pass this target value to the smoothScroll function in task3.js
    • Remeber to cancel the default event when the link is pressed!
